# 네이버 블로그 자동 포스팅 프로그램

네이버 블로그에 자동으로 포스팅할 수 있는 GUI 프로그램입니다.

## 주요 기능

- ✅ GUI 기반 사용자 친화적 인터페이스
- ✅ 네이버 자동 로그인
- ✅ 블로그 글 자동 작성
- ✅ 제목, 본문, 카테고리 설정
- ✅ 실시간 상태 표시

## 설치 방법

### 1. 필수 라이브러리 설치

```bash
pip install -r requirements.txt
```

또는 개별 설치:

```bash
pip install selenium webdriver-manager pyperclip
```

### 2. Chrome 브라우저 설치

이 프로그램은 Chrome 브라우저를 사용합니다. Chrome이 설치되어 있지 않다면 설치해주세요.

## 사용 방법

### 1. 프로그램 실행

```bash
python naver_blog_gui.py
```

### 2. 로그인

1. 네이버 아이디와 비밀번호를 입력합니다.
2. "로그인" 버튼을 클릭합니다.
3. 캡챠가 나타나면 수동으로 해결합니다.
4. 로그인 성공 시 "로그인 완료" 메시지가 표시됩니다.

### 3. 블로그 포스팅

1. 제목을 입력합니다.
2. 내용을 작성합니다.
3. 카테고리를 선택합니다 (기본값: 일상).
4. 공개 설정을 선택합니다.
5. "포스팅하기" 버튼을 클릭합니다.
6. 자동으로 글쓰기 페이지가 열리고 내용이 입력됩니다.
7. 수동으로 발행 버튼을 눌러 포스팅을 완료합니다.

## 주의사항

⚠️ **보안**: 
- 프로그램 실행 시에만 아이디와 비밀번호를 입력하세요.
- 소스 코드에 계정 정보를 저장하지 마세요.

⚠️ **네이버 정책**: 
- 이 프로그램은 교육 목적으로 제작되었습니다.
- 자동화 프로그램 사용은 네이버 이용약관을 위반할 수 있습니다.
- 과도한 사용은 계정 제재를 받을 수 있습니다.

⚠️ **캡챠**: 
- 네이버는 보안을 위해 캡챠를 사용합니다.
- 캡챠가 나타나면 수동으로 해결해야 합니다.

⚠️ **발행 버튼**: 
- 자동 발행은 구현되지 않았습니다.
- 포스팅 작성 후 수동으로 발행 버튼을 눌러주세요.

## 기술 스택

- **Python 3.x**: 프로그래밍 언어
- **Tkinter**: GUI 프레임워크
- **Selenium**: 웹 자동화
- **WebDriver Manager**: ChromeDriver 자동 관리
- **Pyperclip**: 클립보드 조작

## 트러블슈팅

### Chrome 버전 오류
- Chrome 브라우저를 최신 버전으로 업데이트하세요.
- webdriver-manager가 자동으로 호환되는 ChromeDriver를 다운로드합니다.

### 로그인 실패
- 아이디와 비밀번호를 다시 확인하세요.
- 캡챠를 정확히 입력했는지 확인하세요.
- 2단계 인증이 설정되어 있다면 해제하거나 별도 처리가 필요합니다.

### 포스팅 오류
- 네이버 블로그 페이지 구조가 변경되었을 수 있습니다.
- 브라우저를 수동으로 조작하여 포스팅을 완료하세요.
